O JAVA UnBBayes utiliza uma técnica para raciocinar por probabilidades em sistemas inteligentes. Através de uma rede probabilística - grafo onde os nós são variáveis de probabilidade que representam um domínio de conhecimento e os arcos indicam relações entre elas -, a partir de certas evidências, podemos estimar probabilidades condicionadas a essas evidências que nos auxiliam na tomada de decisão. Esse cálculo é chamado de inferência probabilística. Com a técnica da árvore de junção, consegue-se uma inferência em redes probabilísticas de grande eficiência.

Para tornar essa técnica de fácil utilização criamos o JAVA UnBBayes que é um sistema visual, interativo e independente de plataforma, que torna possível a edição, compilação das redes, entrada de evidências e o raciocínio probabilístico.
