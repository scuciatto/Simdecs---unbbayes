Para abrir o módulo de aprendizagem, basta clicar no botão que possui este ícone .

A aprendizagem de uma rede bayseana é feita a partir de um banco de casos. O banco de

casos contém variáveis e as variáveis possuem um ou mais estados. Um caso é formado 

por um definido estado para cada variável. 

BANCO DE CASOS

VisitAsia Tuberculosis Smoking Cancer TbOrCa XRay Bronchitis Dyspnea
NoVisit Absent Smoker Absent False Normal Present Present
NoVisit Absent Smoker Absent False Normal Absent Absent
NoVisit Absent Smoker Absent False Normal Absent Absent
NoVisit Absent NonSmoker Absent False Normal Absent Present

A primeira linha do banco de casos sempre indica as variáveis do banco. Cada linha a seguir 

corresponde a um determinado caso onde cada variável assume um estado.Um banco de casos

 pode ser ainda compactado onde uma determinada variável indica quantas vezes um determinado

caso se repete no banco.

O aprendizado de uma rede bayseana é feito descobrindo-se os pais de cada variável. Os pais de uma determinada variável é descoberto através de vários tipos de algoritmos.
