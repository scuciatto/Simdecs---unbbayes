Setting the Preferences

There are some options possible to be set by the user. They are relative to gain ratio, prunning and verbosity.

To set the preferences:

  1. Click Learning > Preferences
  2. Set the preferences
  3. Click Ok

The available preferences are:

  • Use gain ratio in learning: use gain ratio instead of gain on learning
  • Prune the tree: prune the tree generated using the confidence level set by the user
  • Confidence level: confidence level used on the tree prunning, in percent (enabled if prune the tree is selected)
  • Verbosity level: level of detail of the learning data shown by the Verbosity tab, from 1 to 4

The options are transient: when the user closes UnbMiner, the options are not maintained.
